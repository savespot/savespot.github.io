Electricity Bill Calculation

Assumptions

  • The IOT device is able to capture unit consumption based on the time period.
  • Day – (05.30 am - 6.30 pm)
  • Peak – (6.30 pm - 10.30 pm)
  • Off Peak – (10.30 pm - 05.30 am)
  • When onboarding, the app should be able to capture SME type (via a drop-down list of SME categories).
  • Assume that at the beginning of the month, electricity unit consumption is zero, No broad forward balance.
  • Based on the total consumption, the app is able to cluster rate-wise and consumption-level-wise.
    • Rate wise
      • Wi1 = Rate 1 – Supply at 400/230V contract demand <= 42kVA
      • Wi2 = Rate 2 – Supply at 400/230V contract demand > 42kVA
      • Wi3 = Rate 3 – Supply at 11 kV & above
    • Consumption Level wise
      • IP 1-1 – for <= 300 kWh/month
      • IP 1-2 – for > 300 kWh/month
      • GP 1-1 – for <= 180 kWh/month
      • GP 1-2 – for > 180 kWh/month

Electricity Bill Prediction

Use LSTM model to predict the electricity Bill

The LSTM model is specifically designed to detect patterns in data sequences, utilizing its unique architecture to retain or discard previous information in short-term memory. This capability allows the LSTM model to identify even longer dependencies in sequences, providing better predictions in the context of multi-time series analysis by modeling relationships between multiple input time series.

Model functionality

  • Forget gate: filter only the needed historical data and keep the rest without using it for prediction.eg: Tax policy change: model predict the electricity bill using the new tax policy value and disregard the old tax rates.
  • Input gate: This gate determines which new information from the current input should be stored in the memory cell
  • Output gate: This gate determines which information from the memory cell should be used to produce the output at the current time step.
  • Cell State: Holds the long-term memory. It stores the information that has been allowed into the network by the input gate and retained by the forget gate.
  • Hidden State: This state holds the short-term memory. It is the output of the LSTM cell and is calculated based on the current input and the previous hidden state.

The Long Short-Term Memory (LSTM) model is a powerful technique for modeling time-series data that involves long-term dependencies. It is designed to remember relevant information from the past time steps and selectively incorporate new information at each step. This allows the model to capture complex patterns and relationships in the data, leading to more accurate predictions and better insights.
Unlike other time series models such as exponential smoothing or APA, the LSTM model can incorporate new changes into the next prediction without requiring the model to be recreated. This is especially beneficial since the stationary nature of the data is often crucial for accurate prediction. Therefore, using the LSTM model can overcome these challenges and produce more accurate predictions.

  • LSTM model - PROS
    • LSTM models have the ability to capture complex and non-linear relationships between variables in a multivariate time series. This is because LSTM models can selectively retain and use information from previous time steps, allowing them to capture the dependencies and relationships between different variables that may be critical for accurate prediction.
    • LSTM models have shown to be effective in making accurate long-term predictions due to their ability to retain information over an extended period. This makes them ideal for forecasting tasks where long-term dependencies are crucial. In contrast to traditional models, LSTMs can capture complex and non-linear relationships between variables in a multivariate time series.
  • LSTM model - CONS
    • LSTM models are computationally expensive as they require a large amount of computational resources, especially when processing a large number of time steps or dealing with a large number of features. This can lead to longer training times and higher hardware requirements.
    • LSTM models have a large number of parameters that need to be estimated during training, which means they require a large amount of data to achieve accurate predictions. Additionally, LSTMs are designed to capture complex and non-linear relationships between variables in a time series, which may require even more data to properly identify and model.
